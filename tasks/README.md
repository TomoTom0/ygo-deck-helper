# タスク管理

このディレクトリは新しい運用ルールに従って整理されています。

トップレベルのファイル:

- `todo.md` - 未着手のタスク
- `wip.md` - 現在作業中のタスク
- `done.md` - 完了したタスク
- `milestone.md` - マイルストーン
- `pending.md` - 検討中/保留

一時的に保存したい詳細な作業ノートや未整理のタスクは `tmp/wip/` に配置してください。

作業フロー:

1. 新しいタスクはまず `todo.md` に追加
2. 作業を開始したら該当項目を `wip.md` に移動し、必要な一時ノートは `tmp/wip/` に置く
3. 完了したら `done.md` に移動
